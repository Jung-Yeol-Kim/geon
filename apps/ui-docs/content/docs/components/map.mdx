# Map 컴포넌트

ODF 지도 라이브러리를 React 컴포넌트로 래핑한 지도 컴포넌트입니다.

## 기본 사용법

<Map />

```tsx
import { Map } from '@/components/map'

export default function Page() {
  return <Map />
}
```

## Props

| 이름 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| initialCenter | [number, number] | [276179.88560667867, 413632.9594010007] | 초기 중심 좌표 |
| initialZoom | number | 1 | 초기 줌 레벨 |
| basemap | string | 'eMapBasic' | 기본 지도 타입 |
| className | string | - | 컨테이너 클래스명 |

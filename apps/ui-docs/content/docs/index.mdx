## 소개

GeOn - UI는 Wavus 솔루션인 ODF를 기반으로 한 **패키지**로 GIS 시스템 개발 시 필요한 기본적인 UI와 Hook을 제공합니다.
React 기반 프로젝트에 원활하게 통합되어 높은 유연성을 제공합니다.

GeOn UI는 다음과 같은 파트로 구성됩니다:

<Card icon={<CpuIcon className="text-purple-300" />} title='Map - UI'>

지도와 연동할 수 있는 다양한 컨트롤, TOC (Table of Contents) 를 포함한 UI를 제공합니다.

</Card>

<Card icon={<PanelsTopLeft className="text-blue-300" />} title='Map - Hooks'>

지도를 제어할 수 있는 Hook을 제공합니다. 개발자는 직접 Hook을 적절한 요소에 연결할 수 있습니다.

## 자동 설치

`create-geon-app`을 사용하여 새 앱을 만듭니다. Node.js 18+가 필요합니다.

<Tabs groupId='package-manager' items={['npm', 'pnpm', 'yarn']}>

<Tab>
```bash tab="npm"
npm create geon-app
```
</Tab>

<Tab>
```bash tab="pnpm"
pnpm create geon-app
```
</Tab>
<Tab>
```bash tab="yarn"
yarn create geon-app
```
</Tab>
</Tabs>

CLI 에서 명령을 실행하면 몇 가지 질문이 나옵니다:

- Map-UI, Map-Hooks 패키지를 설치할까요?
- Tailwind CSS를 구성할까요?
- ESLint 구성을 추가할까요?
- 종속성을 설치할까요?

그 후, 새로운 GeOn 앱이 초기화됩니다.
이제 GeOn - UI를 사용할 준비가 되었습니다.


### 즐기세요!

프로젝트를 생성하였다면 다음을 볼 수 있습니다:

```tsx title="app/page.tsx"
// 두가지 방식 중 선택 필요
import { Map } from '@/components/map-ui';  // CLI 설치한 경우   
import { Map } from '@geon/map-ui';         // NPM 설치한 경우

export default function Page() {
  return (
    <main className="flex flex-1 flex-col justify-center text-center">
      <Map />
    </main>
  );
}
```

앱을 개발 모드로 실행하고 http://localhost:3000/ 에서 지도를 확인하세요.

```package-install
npm run dev
```

### 탐색

프로젝트에서 다음을 볼 수 있습니다:

- `app/globals.css`: Map Container 의 기본 스타일이 정의되어 있습니다. 
```css
.odf-map {
  width: 100%;
  height: 100%;
}
```

- `app/componets/map.tsx`: Map-Hooks 와 통합된 배경지도 컨트롤을 추가한 예시를 보여줍니다.
```tsx
export function Page() {
  return (
    ...
  )
}
```

## FAQ

자주 묻는 질문입니다.

<Accordions>
    <Accordion id='customise' title='UI를 어떻게 커스터마이징하나요?'>

GeOn UI는 테마, 레이아웃 및 컴포넌트를 제공합니다.

디자인 시스템은 Tailwind CSS로 구축되었으며, [테마](/docs/ui/theme)를 참조하세요.

컴포넌트는 문서를 향상시키기 위한 사전 제작된 요소로, `style` 및 `className`과 같은 노출된 props로 커스터마이징할 수 있습니다.

레이아웃은 문서의 필수 요소로, 탐색 및 기본 기능을 제공합니다.
여러 가지 옵션을 제공하여 커스터마이징할 수 있으며, [레이아웃](/docs/ui/layouts)을 참조하세요.

GeOn CLI는 프로젝트에 GeOn 컴포넌트를 설치하는 방법을 제공하여 코드로 완전히 커스터마이징할 수 있습니다.

```package-install
@geon/cli --save-dev
```

```bash
pnpm geon add
```

이는 Shadcn UI와 매우 유사합니다.
